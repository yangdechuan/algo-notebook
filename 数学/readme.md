## 斐波那契数列
f(0) = 0  
f(1) = 1  
f(n) = f(n - 1) + f(n - 2)  
方法一：动态规划  
时间复杂度：O(n)  
dp(n) = dp(n - 1) + dp(n - 2)  
方法二：矩阵快速幂  
时间复杂度：O(logn)，但是隐含的时间常数较大。

## 约瑟夫问题
n个人编号0,...,n-1围成一圈，顺时针从1开始报数，每次报到m的人被杀掉并移出去，
然后从下一个人开始报数，直到剩余一个人；求最后活着的人的编号。  
解：  
dp(i)表示一共i个人最后活着的人的编号  
dp(1) = 0  
dp(i) = (dp(i - 1) + m) % i
